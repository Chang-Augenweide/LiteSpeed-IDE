# LiteSpeed-IDE Roadmap

## 🚀 开发路线图

本文档描述 LiteSpeed-IDE 的开发计划和里程碑。

---

## Phase 1: MVP（基础功能）- 2026-02-16 ~ 2026-05-16

**目标**: 可用的基础编辑器，支持文件编辑、终端、基础 Git 操作

### 任务列表

- [x] 项目设计与文档
  - [x] 技术选型（Rust + Tauri + CodeMirror 6）
  - [x] UI/UX 设计指南
  - [x] 架构文档

- [ ] 基础窗口系统
  - [ ] 主窗口框架（Tauri）
  - [ ] 菜单栏（File, Edit, View, Help）
  - [ ] 状态栏（Git 状态、行/列、编码）

- [ ] 文件树
  - [ ] 文件树渲染（虚拟滚动）
  - [ ] 文件/文件夹创建、重命名、删除
  - [ ] 拖拽支持
  - [ ] 图标文件类型识别

- [ ] 编辑器
  - [ ] CodeMirror 6 集成
  - [ ] 语法高亮（Tree-sitter WASM）
  - [ ] 多标签页支持
  - [ ] 大文件虚拟滚动
  - [ ] 搜索替换（跨文件）

- [ ] 文件系统
  - [ ] 文件打开/保存
  - [ ] 自动保存（可选）
  - [ ] 文件监听（外部修改）
  - [ ] 快速打开（Ctrl+P）

- [ ] 终端
  - [ ] 终端嵌入（xterm.js / 系统 Webview）
  - [ ] 多标签页支持
  - [ ] Shell 自动检测
  - [ ] 大小调整（拖动）

- [ ] Git 集成（基础）
  - [ ] Git 状态显示
  - [ ] Add / Commit / Push / Pull
  - [ ] 简化 Diff 视图
  - [ ] 分支切换

**完成标准**:
- ✅ 可以打开/编辑/保存文件
- ✅ 可以使用终端执行命令
- ✅ 可以进行基础 Git 操作

## Phase 2: 插件系统 - 2026-05-16 ~ 2026-07-16

**目标**: 可扩展的插件系统，提供本地插件市场

### 任务列表

- [ ] WASM 插件 Runtime
  - [ ] WASM 编译与加载
  - [ ] 插件 API 设计（Rust ↔ WASM）
  - [ ] 权限管理系统
  - [ ] 沙箱隔离

- [ ] 插件管理器
  - [ ] 插件安装/卸载
  - [ ] 插件更新检查
  - [ ] 插件启用/禁用
  - [ ] 插件热加载

- [ ] 插件开发文档
  - [ ] 插件 API 文档
  - [ ] 开发者指南
  - [ ] 插件示例（3 个）

- [ ] 内置插件
  - [ ] Python LSP 插件
  - [ ] Go LSP 插件
  - [ ] Rust LSP 插件

- [ ] 本地插件市场
  - [ ] 插件列表展示
  - [ ] 插件详情页
  - [ ] 安装/更新按钮
  - [ ] 插件评分（v2.x）

**完成标准**:
- ✅ 可以加载 WASM 插件
- ✅ 插件可以访问受限 API
- ✅ 3 个内置插件可用

## Phase 4: 优化与完善 - 2026-09-16 ~ 2026-10-16

**目标**: 性能调优，UI/UX 完善，准备 v1.0.0 Release

### 任务列表

- [ ] 性能优化
  - [ ] 启动速度（目标 < 0.5s）
  - [ ] 内存占用（目标 < 120MB）
  - [ ] 大文件处理（目标 100MB < 300ms）
  - [ ] 搜索性能（目标 1M 行 < 200ms）

- [ ] UI/UX 完善
  - [ ] 深色主题完善
  - [ ] 高分辨率支持（HiDPI）
  - [ ] 多语言支持（i18n）
  - [ ] 快捷键自定义

- [ ] 文档
  - [ ] 用户手册
  - [ ] API 文档
  - [ ] 插件开发指南
  - [ ] FAQ

- [ ] 测试
  - [ ] 单元测试覆盖率 > 80%
  - [ ] 集成测试
  - [ ] 性能基准测试
  - [ ] 手动测试

- [ ] 发布准备
  - [ ] 打包脚本
  - [ ] 安装包生成（Windows/macOS/Linux）
  - [ ] Release Notes
  - [ ] 更新日志

**完成标准**:
- ✅ 所有核心指标达标
- ✅ 文档完整
- ✅ 性能测试通过
- ✅ v1.0.0 正式发布
